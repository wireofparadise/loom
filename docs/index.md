# Introduction

## What is Loom?

**Loom** is a game development framework for Roblox that includes component lifecycle management, logging and networking.

## Why does it exist?

I created Loom because I spent weeks searching for a stack that would allow clean architecture patterns but ultimately failed.

It was inspired by [Knit](https://sleitnick.github.io/Knit/) but because it is no longer maintained, and is built in a way that blocks autocompletion in language servers, I opted not to use it.

## Is Loom available for TypeScript?

No, Loom is a framework designed specifically for Luau and it will not be supported for `roblox-ts`.
