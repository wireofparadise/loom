# Introduction

**Loom** is a lifecycle management framework for Roblox.

It is developed because a popular alternative [Knit](https://sleitnick.github.io/Knit/) is no longer maintained.

If you think that anything is missing or incorrect, feel free to open an [issue](https://github.com/wireofparadise/loom/issues) or submit a [pull request](https://github.com/wireofparadise/loom/pulls) on GitHub.
