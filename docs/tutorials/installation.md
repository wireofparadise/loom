# Installation

=== "wally"
    Add the following line to your `wally.toml`:

    ```toml
    Loom = "wireofparadise/loom@1.0.0"
    ```

    And then reinstall your packages, using the following command:

    ```sh
    wally install
    ```

    !!! note "Wally package types"
        For wally, I recommend using [wally-package-types](https://github.com/JohnnyMorganz/wally-package-types) to foward exported types to the module script generated by wally. This allows you to reference Luau types used by Spaceship.
    
=== "rbxm"
    Installing Loom as a standalone roblox model:

    Navigate to [the latest release on GitHub](https://github.com/wireofparadise/loom/releases/latest), download the `Loom.rbxm` or `Loom.rbxmx` file, and insert it into your place.
